# Bewin Babu — Portfolio

A modern, high-performance portfolio built with **React**, **TypeScript**, **Tailwind CSS**, and **Three.js**.

## Features

- **Light & Dark themes** with smooth toggle (sun/moon button)
- **3D animated backgrounds** — Ghibli-inspired (light) / Cyberpunk (dark)
- **Fully responsive** — works on all devices
- **Lazy-loaded sections** for fast initial page load
- **Code-split vendor chunks** (Three.js, Framer Motion, React)

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS
- Three.js / React Three Fiber / Drei
- Framer Motion

## Getting Started

```sh
npm install
npm run dev
```

## Building for Production

```sh
npm run build
npm run preview
```
